##Features

##To Run Tests
```
mvn test
```

##To Install
```
mvn install
cp EkstaziJenkinsPlugin.hpi $JENKINS_HOME/plugins
http://yourjenkinsserver:8080/jenkins/reload
```

##To Debug
```
mvn hpi:run
```
